HTML Embed Template

<!-- Google Map Section --> <section class="map-section"> <h2>מיקום המשרד</h2> <div class="map-container"> <iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!..." width="100%" height="400" style="border:0; border-radius: 12px;" allowfullscreen="" loading="lazy" referrerpolicy="no-referrer-when-downgrade" title="משרד מרום גולי שמאות מקרקעין"> </iframe> </div> <p> <strong>כתובת:</strong> הארד 9, רמת החייל, תל אביב <a href="https://waze.com/ul?q=הארד 9 רמת החייל" target="_blank"> נווט עם Waze </a> </p> </section>

Schema Update Required

Add to existing LocalBusiness schema:

"geo": { "@type": "GeoCoordinates", "latitude": "32.1234", "longitude": "34.8234" }, "hasMap": "https://maps.google.com/?q=הארד+9+רמת+החייל+תל+אביב"
2
HIGH PRIORITY
Customer Reviews Section

The client has an active Google My Business profile. Adding a customer reviews section with proper Review schema markup enables rich snippets in search results and increases user trust.

Implementation Checklist

  • Obtain client's Google My Business profile URL
  • Collect 3-5 approved customer reviews to display
  • Create reviews section component with RTL support
  • Design review cards with star ratings (gold color #FFD700)
  • Add Google rating badge showing overall score
  • Implement "Write a Review" CTA button linking to GMB
  • Add AggregateRating schema
  • Add individual Review schemas (minimum 3)
  • Validate schema at Google Rich Results Test

Review Card HTML Template

<section class="reviews-section" dir="rtl"> <h2>מה לקוחותינו אומרים</h2> <!-- Google Rating Badge --> <div class="google-rating"> <img src="/google-logo.png" alt="Google"> <span class="rating">4.9</span> <span class="stars">★★★★★</span> <span>(XX ביקורות)</span> </div> <!-- Review Cards Grid --> <div class="reviews-grid"> <div class="review-card"> <div class="review-header"> <div class="avatar">א</div> <div> <strong>שם הלקוח</strong> <span class="stars">★★★★★</span> </div> </div> <p>"תוכן הביקורת כאן..."</p> <span class="date">לפני 3 חודשים</span> </div> </div> <!-- CTA Button --> <a href="GMB_REVIEW_LINK" class="review-cta"> כתבו לנו ביקורת בגוגל → </a> </section>

Required Schema Markup

{ "@context": "https://schema.org", "@type": "LocalBusiness", "name": "מרום גולי שמאות מקרקעין בע״מ", "aggregateRating": { "@type": "AggregateRating", "ratingValue": "4.9", "reviewCount": "XX", "bestRating": "5", "worstRating": "1" }, "review": [ { "@type": "Review", "author": { "@type": "Person", "name": "לקוח מאומת" }, "reviewRating": { "@type": "Rating", "ratingValue": "5" }, "reviewBody": "Review text here...", "datePublished": "2025-XX-XX" } ] }

Important Note

Reviews displayed must be authentic and from the client's actual GMB profile. Get written approval from the client before publishing any customer reviews on the website.

3
HIGH PRIORITY
Social Media Links

The website currently has no social media links - this is a significant missing element. Social signals contribute to brand authority and provide additional channels for client engagement.

Required Platforms

Platform Priority Purpose Action Required
LinkedIn Critical Professional network, B2B leads Get profile URL from client
Facebook High Local audience, case studies Get page URL from client

Implementation Checklist

  • Confirm social media profile URLs with client
  • Add social icons to header (small, right-aligned)
  • Create social media section in footer
  • Use SVG icons for optimal quality
  • Add hover states with platform brand colors
  • Ensure links open in new tab (target="_blank")
  • Add rel="noopener noreferrer" for security
  • Add aria-label for accessibility
  • Update Organization schema with sameAs URLs

Header Implementation

<!-- Social Links - Header --> <div class="social-links-header"> <a href="LINKEDIN_URL" target="_blank" rel="noopener noreferrer" aria-label="LinkedIn"> <!-- LinkedIn SVG Icon --> <svg width="20" height="20" viewBox="0 0 24 24" fill="currentColor"> <path d="M20.447 20.452h-3.554v-5.569c0-1.328..."></path> </svg> </a> <a href="FACEBOOK_URL" target="_blank" rel="noopener noreferrer" aria-label="Facebook"> <!-- Facebook SVG Icon --> </a> </div>

Footer Implementation

<!-- Social Links - Footer --> <div class="footer-social"> <p>עקבו אחרינו:</p> <div class="social-icons"> <a href="LINKEDIN_URL" target="_blank" rel="noopener"> LinkedIn </a> <a href="FACEBOOK_URL" target="_blank" rel="noopener"> Facebook </a> </div> </div>

Schema Update Required

Add sameAs to existing Organization schema:

{ "@type": "Organization", "name": "מרום גולי שמאות מקרקעין בע״מ", "sameAs": [ "https://www.linkedin.com/in/marom-guli-XXXXX/", "https://www.facebook.com/maromguli.appraiser/" ] }

Action Required from Client

Before implementation, confirm the following with the client:

  • Do they have a LinkedIn profile? If not, recommend creating one (essential for B2B)
  • Do they have a Facebook business page?
  • Get exact URLs for all active profiles
