1.4 BreadcrumbList Schema (כל העמודים הפנימיים)

TASK #4: הטמעת BreadcrumbList Schema עדיפות בינונית

עמודים: כל העמודים הפנימיים (דפי שירות, מאמרים)

דוגמה לדף עובדים זרים:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "דף הבית",
      "item": "https://www.manpower.co.il/"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "מעסיקים",
      "item": "https://www.manpower.co.il/employers/employers"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "גיוס עובדים זרים"
    }
  ]
}
</script>
חשוב: הפריט האחרון (העמוד הנוכחי) לא מקבל שדה "item" - רק "name".

2. תיקוני Meta Tags

2.1 דפי שירות - Meta Descriptions חסרים

TASK #6: הוספת Meta Description לדף פיננסים עדיפות גבוהה

עמוד: /specializations/finance

מצב נוכחי: חסר לחלוטין

קוד לשינוי:

<!-- Title Tag (לשנות) -->
<title>גיוס והשמה בתחום הפיננסים | מומחי גיוס פיננסי | מנפאואר ישראל</title>

<!-- Meta Description (להוסיף) -->
<meta name="description" content="מחפשים טאלנטים פיננסיים? מנפאואר מתמחה בגיוס והשמה בתחום הפיננסים - רואי חשבון, כלכלנים, אנליסטים ומנהלי כספים. 65 שנות ניסיון בשוק הישראלי.">

<!-- OG Tags (להוסיף/לעדכן) -->
<meta property="og:title" content="גיוס והשמה בתחום הפיננסים | מנפאואר ישראל">
<meta property="og:description" content="מחפשים טאלנטים פיננסיים? מנפאואר מתמחה בגיוס והשמה בתחום הפיננסים - רואי חשבון, כלכלנים, אנליסטים ומנהלי כספים.">
TASK #7: תיקון דף מדעי החיים + שגיאת כתיב עדיפות גבוהה

עמוד: /specializations/LIFE-SIENCE

בעיות:

  • שגיאת כתיב ב-URL: SIENCE במקום SCIENCE
  • Meta Description חסר
  • Title גנרי מדי
אזהרה: תיקון ה-URL דורש הגדרת 301 Redirect מה-URL הישן לחדש! 
<!-- Title Tag (לשנות) -->
<title>גיוס והשמה במדעי החיים | Life Sciences Recruitment | מנפאואר ישראל</title>

<!-- Meta Description (להוסיף) -->
<meta name="description" content="מנפאואר מתמחה בגיוס טאלנטים למדעי החיים - ביוטכנולוגיה, פארמה, מכשור רפואי ומחקר קליני. גיוס מקצועי של רוקחים, אחיות, טכנאי מעבדה ומנהלים בכירים.">

<!-- 301 Redirect נדרש (בקובץ .htaccess או Next.js config) -->
Redirect 301 /specializations/LIFE-SIENCE /specializations/life-science

3. תיקונים טכניים

3.1 תיקון שגיאות 404

TASK #14: בדיקת ותיקון 5 מאמרים עם 404 עדיפות בינונית

מאמרים מושפעים:

מאמר URL שמחזיר 404 פעולה נדרשת
המדריך לחיפוש עבודה /article/חיפוש-עבודה בדוק מיפוי URL במסד נתונים
שדרוג קורות חיים /article/איך-לשדרג-קורות-חיים... בדוק מיפוי URL
ירידי תעסוקה /article/ירידי-תעסוקה... בדוק מיפוי URL
אתגרים בגיוס בכירים /article/האתגרים-בגיוס... בדוק מיפוי URL
בחירת מקום עבודה /article/מה-כדאי-לקחת... בדוק מיפוי URL
הערה: ככל הנראה הבעיה היא בקיצור ה-URL (truncation) או בשינוי slug. יש לבדוק את ה-URL המקורי במערכת ולהגדיר redirects.
