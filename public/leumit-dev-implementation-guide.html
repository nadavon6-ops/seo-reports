1.1 קריטי מוכן להעתקה BreadcrumbList Schema - הטמעה דינמית

פרטתיאור
מה זהסימון מבנה הניווט (פירורי לחם) לגוגל
למה חשובמשפר הצגה בתוצאות החיפוש, מקל על משתמשים
היכן ליישםכל העמודים באתר (דינמי)
מיקום ב-UmbracoTemplate Level - לפני </head>
כללי יישום:
  • כל עמוד צריך BreadcrumbList ייחודי - נבנה דינמית מהמבנה
  • הפריט האחרון (העמוד הנוכחי) - ללא שדה "item" (רק name)
  • position חייב להיות רציף: 1, 2, 3...
  • כל ה-URLs חייבים להיות מלאים עם https://
דוגמה 1 - עמוד מחלה (חצבת):
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "BreadcrumbList", "itemListElement": [ { "@type": "ListItem", "position": 1, "name": "דף הבית", "item": "https://www.leumit.co.il/" }, { "@type": "ListItem", "position": 2, "name": "מחלות", "item": "https://www.leumit.co.il/diseases/" }, { "@type": "ListItem", "position": 3, "name": "חצבת" } ] } </script>
דוגמה 2 - עמוד ביטוח משלים:
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "BreadcrumbList", "itemListElement": [ { "@type": "ListItem", "position": 1, "name": "דף הבית", "item": "https://www.leumit.co.il/" }, { "@type": "ListItem", "position": 2, "name": "ביטוחי משלים", "item": "https://www.leumit.co.il/insurance-policies/" }, { "@type": "ListItem", "position": 3, "name": "תכנית כסף וזהב" } ] } </script>
דוגמה 3 - עמוד צור קשר:
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "BreadcrumbList", "itemListElement": [ { "@type": "ListItem", "position": 1, "name": "דף הבית", "item": "https://www.leumit.co.il/" }, { "@type": "ListItem", "position": 2, "name": "צור קשר" } ] } </script>
שימו לב: הפריט האחרון (העמוד הנוכחי) מכיל רק "name" ללא "item"!

  • או: עריכה ישירה בתבנית עם Razor
<!-- ב-head section של התבנית --> <title>@(Model.HasValue("pageTitle") ? Model.Value("pageTitle") : Model.Name) | לאומית</title> <meta name="description" content="@Model.Value("metaDescription")">

5 בדיקות צ'קליסט בדיקה לפני העלאה

לכל Schema שמעלים:

  • בדיקה ב-validator.schema.org - אין שגיאות
  • בדיקה ב-Google Rich Results Test - תוצאה חיובית
  • כל ה-URLs הם https:// מלאים
  • אין שדות ריקים או עם ערך null
  • כל התווים המיוחדים מקודדים נכון (גרשיים, אמפרסנד)

לכל Meta Tag שמעדכנים:

  • Meta Title: עד 60 תווים
  • Meta Description: עד 160 תווים
  • אין תווים מיוחדים שעלולים להישבר
  • הטקסט קריא ומזמין לחיצה

בדיקה סופית לאחר העלאה:

  • צפייה בקוד מקור של העמוד - Schema מופיע
  • בדיקה שהעמוד נטען כרגיל
  • בדיקה ב-Google Search Console → URL Inspection
  • בקשת אינדוקס מחדש (Request Indexing)
כלים מומלצים:
  • Schema Validator
  • Rich Results Test
  • Google Search Console
