1 עדיפות קריטית הטמעת Schema Markup

זוהי המשימה החשובה ביותר. הטמעת Schema מאפשרת הצגת Rich Results בגוגל, הגדלת CTR, ותפיסת Featured Snippets. יש לבצע לפני כל משימה אחרת.

סדר סוג Schema היכן ליישם עדיפות

1.2 Organization עמוד הבית בלבד קריטי

1.1 קריטי BreadcrumbList Schema - הטמעה כלל-אתרית

פרטתיאור
מה זהסימון מבנה הניווט (פירורי לחם) לגוגל
למה חשובמשפר הצגה בתוצאות החיפוש, מקל על משתמשים
היכן ליישםכל העמודים באתר
סוג משימהטכני
כללי יישום:
  • כל עמוד צריך BreadcrumbList ייחודי משלו
  • הפריט האחרון (העמוד הנוכחי) - ללא שדה "item" (רק name)
  • position חייב להיות רציף: 1, 2, 3...
  • כל ה-URLs חייבים להיות מלאים (כולל https://)
  • שמות בעברית
תבנית קוד:
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "BreadcrumbList", "itemListElement": [ { "@type": "ListItem", "position": 1, "name": "דף הבית", "item": "https://www.leumit.co.il/" }, { "@type": "ListItem", "position": 2, "name": "מחלות", "item": "https://www.leumit.co.il/diseases/" }, { "@type": "ListItem", "position": 3, "name": "חצבת" } ] } </script>
שימו לב: הפריט האחרון (העמוד הנוכחי) מכיל רק "name" ללא "item"!

1.3 קריטי FAQPage Schema - עמודי תוכן רפואי

פרטתיאור

למה חשובFeatured Snippets, תצוגה מורחבת בתוצאות
היכן ליישםכל עמוד שמכיל בלוק FAQ
סוג משימהמשולב (טכני + תוכן)
עמודים ליישום - לפי סדר עדיפות:
סדר עמוד URL מספר שאלות
1 חצבת /diseases/measles/ 10 שאלות
2 שפעת /diseases/influenza-information-about-the-disease/ 12 שאלות
3 כסף וזהב /insurance-policies/leumit-silver-and-gold/ 8 שאלות

תבנית קוד - דוגמה לעמוד חצבת:
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "FAQPage", "mainEntity": [ { "@type": "Question", "name": "מה הם הסימפטומים הראשונים של חצבת?", "acceptedAnswer": { "@type": "Answer", "text": "הסימפטומים הראשונים כוללים חום גבוה (מעל 38.5), נזלת, שיעול יבש, עיניים אדומות ודולקות, ולאחר 3-4 ימים מופיעה פריחה אדומה שמתחילה בפנים ומתפשטת לכל הגוף." } }, { "@type": "Question", "name": "האם חצבת מסוכנת למבוגרים?", "acceptedAnswer": { "@type": "Answer", "text": "כן, חצבת מסוכנת יותר למבוגרים מאשר לילדים. סיבוכים כמו דלקת ריאות ודלקת המוח שכיחים יותר. מבוגרים שלא חוסנו או לא חלו בילדות צריכים להתחסן." } }, { "@type": "Question", "name": "תוך כמה זמן צריך להתחסן אחרי חשיפה לחולה חצבת?", "acceptedAnswer": { "@type": "Answer", "text": "יש להתחסן תוך 72 שעות מהחשיפה לחולה חצבת. חיסון לאחר חשיפה יכול למנוע את המחלה או להפחית את חומרתה." } }, { "@type": "Question", "name": "האם אישה בהריון יכולה להתחסן נגד חצבת?", "acceptedAnswer": { "@type": "Answer", "text": "לא, חיסון MMR אינו מותר בהריון כיוון שהוא חיסון חי מוחלש. נשים שאינן חסינות צריכות להתחסן לפני ההריון ולהמתין חודש לפני הכניסה להריון." } }, { "@type": "Question", "name": "כמה זמן חולה חצבת מדביק?", "acceptedAnswer": { "@type": "Answer", "text": "חולה חצבת מדביק מ-4 ימים לפני הופעת הפריחה ועד 4 ימים אחריה - סה\"כ כ-8 ימים. החצבת מדבקת מאוד - עד 90% מהאנשים שאינם מחוסנים ונחשפו יחלו." } } ] } </script>
שאלות מומלצות לכל עמוד:

עמוד שפעת:

  • מה ההבדל בין שפעת להצטננות?
  • מי צריך להתחסן נגד שפעת?
  • האם חיסון השפעת יכול לגרום לשפעת?
  • מתי צריך לפנות לרופא כששיש שפעת?
  • כמה זמן נמשכת שפעת?
  • האם שפעת מסוכנת?
  • איך מונעים הדבקה בשפעת?
  • האם ניתן להתחסן בהריון?

עמוד כסף וזהב:

  • כמה עולה תכנית כסף/זהב?
  • מה ההבדל בין תכנית כסף לתכנית זהב?
  • האם שווה לשדרג מכסף לזהב?
  • מה כלול בתכנית כסף?
  • מה כלול בתכנית זהב?
  • איך מצטרפים לתכנית משלים?
כללים חשובים:
  • כל שאלה חייבת לסיים בסימן שאלה
  • תשובות צריכות להיות ברורות ומלאות
  • אל תכללו שאלות כפולות
  • התשובה ב-Schema חייבת להתאים לתשובה שמופיעה בעמוד
היכן לא ליישם FAQPage Schema:
  • עמוד הבית (Homepage)
  • עמודי חיפוש (Search Results)
  • עמודי קטגוריות ללא תוכן FAQ
  • עמודי התחברות/הרשמה

1.4 גבוהה MedicalWebPage Schema - מדריכים רפואיים

פרטתיאור
מה זהסימון עמוד כתוכן רפואי
למה חשובעוזר לגוגל לסווג נכון תוכן רפואי
היכן ליישםמדריכי מחלות, מידע רפואי
סוג משימהטכני
תבנית קוד:
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "MedicalWebPage", "name": "חצבת - מידע, תסמינים וחיסון", "description": "מידע מקיף על מחלת החצבת - תסמינים, דרכי הדבקה, חיסון ומניעה", "url": "https://www.leumit.co.il/diseases/measles/", "inLanguage": "he-IL", "datePublished": "2024-01-01", "dateModified": "2026-01-20", "publisher": { "@type": "Organization", "name": "לאומית שירותי בריאות", "@id": "https://www.leumit.co.il/#organization" }, "medicalAudience": { "@type": "MedicalAudience", "audienceType": "Patient" } } </script>

הנחיות יישום כלליות - Schema

מיקום הקוד:
  • כל ה-Schema צריך להיות בתוך תג <script type="application/ld+json">
  • מיקום מומלץ: לפני סגירת </head> או לפני סגירת </body>
  • ניתן לשלב מספר Schemas באותו עמוד (כל אחד בתג script נפרד)
בדיקה ואימות:
  • לפני העלאה - לבדוק ב: schema.org/validator
  • אחרי העלאה - לבדוק ב: Google Rich Results Test
  • לוודא שאין שגיאות או אזהרות

3 מידע בלבד פערי תוכן מול מתחרים

הבהרה: סעיף זה הוא אינפורמטיבי בלבד ואינו מצריך פעולה מצוות הפיתוח בשלב זה. המידע מוצג להבנת ההקשר העסקי.

סיכום פערי התוכן

בהשוואה לקופות החולים המתחרות (כללית, מכבי, מאוחדת), לאומית חסרים מספר נושאי תוכן:

נושא מצב בלאומית מצב אצל מתחרים
סוכרת (מדריך מקיף) עמוד סדנה אחד 2-7 עמודים לכל קופה
בריאות הנפש אזכור בודד 8+ עמודים ייעודיים
חיסונים מינימלי 7+ מדריכים + לוח חיסונים
מחשבון הריון לא קיים קיים במכבי ובמאוחדת
מחשבון BMI לילדים לא קיים קיים במכבי
"עולמות" תוכן לא קיים 8 האבים במכבי ("עולמות במכבי")
לתשומת לב: יצירת תוכן חדש היא משימה של צוות התוכן/שיווק ותתואם בנפרד. אין פעולה נדרשת מצוות הפיתוח בשלב זה.
