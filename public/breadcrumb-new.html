Copy & Paste Schemas for 4 Sites Missing BreadcrumbList

These sites have NO breadcrumb schema - implement from scratch

Breadcrumb Schema Rules

  • Dynamic Per Page: Each page must have its OWN breadcrumb path. These are TEMPLATES - adapt for each page.
  • Last Item No URL: The current page (last item) should NOT have an "item" property.
  • Sequential Positions: Position values must be 1, 2, 3... with no gaps.
  • Absolute URLs: All URLs must be complete (start with https://).

After Implementation - Validation Steps

  • Step 1: Test each page at Google Rich Results Test
  • Step 2: Verify no errors in Google Search Console → Enhancements → Breadcrumbs
  • Step 3: Check multiple page types (homepage, category, detail pages) to confirm dynamic implementation
  • Step 4: Ensure the last breadcrumb item has NO "item" property
  • Step 5: Monitor for 2-4 weeks for breadcrumb appearance in search results
