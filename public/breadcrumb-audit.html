Breadcrumb Schema Rules Reminder

1. Dynamic Per Page: Each page needs its OWN breadcrumb (Home → Category → Page)

2. Last Item No URL: The current page should NOT have an "item" property

3. Sequential Positions: Must be 1, 2, 3... without gaps

4. Absolute URLs: All URLs must start with https://
