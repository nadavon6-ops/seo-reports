הטמעה ב-Umbraco CMS:

יש להוסיף את סכמת ה-Breadcrumb דרך מאפייני ה-Document Type או ליצור Partial View שמייצר JSON-LD על בסיס היררכיית התוכן. לחלופין, ניתן להזריק דרך ה-Master Template של האתר.

הוראות הטמעה

  1. העתקת הקוד: לחצו על כפתור "Copy Schema" להעתקת קוד ה-Schema
  2. מיקום ההטמעה: הדביקו את הקוד בתוך תגית <head> בכל דפי האתר
  3. Umbraco CMS: הוסיפו דרך Partial View או Master Template
  4. בדיקת תקינות: לאחר ההטמעה, בדקו ב-Google Rich Results Test
  5. חשוב: יש להטמיע את כל הסכמות - Organization, WebSite ו-BreadcrumbList (לכל דף)
