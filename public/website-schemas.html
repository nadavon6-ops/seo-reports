WebSite Schema Best Practices Applied

  • @context - Schema.org context included
  • @id - Unique identifier with /#website
  • name - Primary site name
  • alternateName - Hebrew/English alternative
  • description - Clear site description
  • url - Canonical homepage URL
  • inLanguage - he-IL for Hebrew sites
  • publisher - Links to Organization schema
  • SearchAction - EntryPoint format with urlTemplate
  • query-input - PropertyValueSpecification format
